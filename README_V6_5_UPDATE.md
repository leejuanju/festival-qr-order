# v6.5 업데이트

## 변경사항

- 홀 화면의 테이블 상세 주문 섹션을 테이블 번호순이 아니라 첫 주문 접수순으로 정렬합니다.
- 홀 화면과 주방 화면에서 주문 단위가 아니라 제공 항목 단위로 체크할 수 있습니다.
- 세트 메뉴는 구성품별로 제공 체크가 가능합니다.
- 새 세트 메뉴를 추가할 때 관리자 화면의 `제공 체크 구성` 입력칸에 구성품을 입력할 수 있습니다.

## 제공 체크 구성 입력 형식

한 줄에 하나씩 아래 형식으로 입력합니다.

```text
메뉴명|수량
```

예시:

```text
닭꼬치|2
염통꼬치|3
하이볼|2
```

비워두면 해당 메뉴 자체가 하나의 제공 체크 항목으로 표시됩니다.

## DB 적용

Supabase SQL Editor에서 다음 파일을 실행합니다.

```text
supabase/v6_5_migration.sql
```

기존 주문 데이터는 삭제하지 않습니다. 새 주문부터 제공 체크 항목이 주문 JSON에 함께 저장됩니다.
