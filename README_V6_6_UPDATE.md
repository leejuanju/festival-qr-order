# v6.6 Update

## 변경 내용

- 주방 주문표에서 기존의 주문 메뉴 목록과 제공 체크 목록이 중복으로 보이던 구조를 제거했습니다.
- 일반 메뉴는 메뉴 행 자체를 누르면 제공 체크가 됩니다.
- 세트 메뉴는 세트 카드 안에 구성품별 체크 버튼이 표시됩니다.
- 홀 화면도 같은 방식의 제공 체크 UI를 사용합니다.
- `+`가 포함된 세트명은 `serve_components`가 없어도 기본적으로 구성품으로 추론합니다.
- 기존 세트 메뉴의 구성품을 보정하는 `supabase/v6_6_migration.sql`을 추가했습니다.

## 세트 메뉴 추가 규칙

관리자 화면의 `제공 체크 구성`에 한 줄씩 입력합니다.

```text
닭꼬치|2
염통볶음|1
하이볼|2
```

비워두면 일반 메뉴로 취급합니다. 메뉴명에 `+`가 포함되어 있으면 자동 분해를 시도하지만, 정확한 운영을 위해 세트는 `제공 체크 구성`을 직접 입력하는 것을 권장합니다.
